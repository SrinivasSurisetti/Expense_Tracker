# 💰 Expense Tracker

A **Full-Stack Expense Tracker Web Application** built using **Angular, Node.js, Express, and MongoDB**.  
The application allows users to **register, login, and manage their personal expenses securely**.

---

## 🚀 Features

- User Registration and Login  
- Secure Authentication using **JWT**  
- Add new expenses  
- Delete expenses  
- View total expenses  
- User-specific expense tracking  

---

## 🛠 Tech Stack

### Frontend
- Angular
- HTML

- TypeScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JSON Web Token (JWT)

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/SrinivasSurisetti/Expense-Tracker.git
cd Expense-Tracker
2️⃣ Backend Setup
cd expense-tracker-backend
npm install
nodemon server.js

Backend runs on:

http://localhost:5000
3️⃣ Frontend Setup
cd expense-tracker-angular
npm install
ng serve

Frontend runs on:

http://localhost:4200
🔐 Authentication Flow

User registers an account

User logs in with credentials

Server generates a JWT Token

Token is used to authorize API requests
